Initialisation:

Source: https://www.cheky.net

Pour créer et démarrer le containers :

	docker-compose up -d

Pour arrêter et supprimer le containers :

	docker-compose down


Pour aller sur l'interface, ouvrer le navigateur firefox/chrome/... et aller sur : http://<@IP>:9080
